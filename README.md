# Call Center Performance Analysis (Power BI)

## Overview
This project presents a Power BI dashboard designed to analyze and improve call center performance. It provides insights into key metrics such as call volume, resolution efficiency, and agent performance.

## Key Objectives
•	Analyse overall call center performance
•	Track call volume trends
•	Measure average resolution time
•	Evaluate agent efficiency
•	Identify areas for operational improvement

## Key Features
* Call Duration Analysis by request type and branch
* Agent-level filtering for performance tracking
* Customer segmentation (account type & tenure)
* Work shift analysis (peak hours identification)
* Branch performance comparison
* Interactive filters and slicers

## Key Insights
* Certain request types contribute to higher call durations
* Call volumes vary across branches
* Peak activity observed in specific shifts
* Checking accounts generate slightly higher support interactions

## Tools & Technologies
* Power BI
* Data Modeling
* Data Visualization
* Excel

## How to Use
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Use filters to explore insights

## Business Impact
* Improves customer service efficiency
* Helps optimize staffing across shifts
* Identifies high-demand service areas
* Supports data-driven decision-making
